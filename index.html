Welcome to my page

This website is made as the second practical work for the Fundamentals of computer engineering course at Universidad Francisco de Vitoria. Here you will find some information about my Fundamentals of Computer Engineering course, as well as some facts about me and some topics I find interesting.

This website is built with the three basic languages of web developing: HTML5, CSS3 and JavaScript.

HTML5 stands for HyperText Markup Language on its fifth version, it is what makes the structure of the webpage, divided into sections or containers. CSS3 stands for Cascade Style Sheet on its third version, a language that allows the website to look just like the developer wants. JavaScript is the only programming language involved in web development, which allows the developer to make the website interactive and adds some practical functionality.
